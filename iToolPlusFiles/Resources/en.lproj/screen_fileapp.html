FileApp the iPhone & iPad File Manager

FileApp is a portable file system that lets you store, view and manage files on the go.

  • Read PDF, Word, Excel, PowerPoint and iWork docs
  • Play music and videos
  • Manage and edit photos
  • Annotate PDF and edit text files
  • Send and manage Email attachments
  • Copy, move, rename, zip files and folders
  • And more...
