FileApp, per gestire file su iPhone e iPad

FileApp è un file system portatile con cui puoi archiviare, visualizzare e gestire i file mentre sei in giro.

  • Leggi file PDF, Word, Excel, PowerPoint e documenti iWork
  • Ascolta musica e guarda i video
  • Gestisci e modifica foto
  • Inserisci commenti e note in file PDF e modifica file di testo
  • Invia e gestisci gli allegati alle e-mail
  • Copia, sposta, rinomina e racchiudi file e cartelle in file ZIP
  • E molto altro...
